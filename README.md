# 浏览器主页

一个简洁、功能丰富的浏览器起始页，集成 Notion、天气、搜索引擎和书签功能。

## 功能特性

- **多搜索引擎支持**: Google、Bing、百度快速切换
- **天气显示**: 自动定位，实时天气信息
- **工作倒计时**: 基于自定义工作时间的智能提醒
- **Notion 集成**: 快速访问 Notion 数据库和工作区
- **书签管理**: 快速访问常用网站
- **主题切换**: 支持深色/浅色模式
- **动态背景**: 几何图形浮动动画效果
- **欢迎语**: 基于时间和上下文的智能问候

## 本地运行

直接用浏览器打开 `index.html` 即可使用。

## 配置

点击页面右上角设置按钮，配置以下选项：

- **Notion API Token**: 用于集成 Notion 功能
- **数据库 ID**: 配置各功能对应的 Notion 数据库
- **页面 URL**: 配置快捷访问链接
- **搜索引擎**: 选择默认搜索引擎
- **天气位置**: 设置天气显示的城市

## 工作时间配置

默认工作时间：
- 上午: 8:00 - 11:20
- 下午: 13:30 - 17:00

可在 JavaScript 代码中修改 `updateWorkCountdown` 函数来自定义。

## 浏览器设置

将此页面设置为浏览器主页：

1. **Chrome**: 设置 → 启动时 → 打开特定网页 → 添加本地文件路径
2. **Firefox**: 选项 → 主页 → 主页和新窗口 → 自定义 → 输入本地文件路径
3. **Safari**: 偏好设置 → 通用 → 主页 → 输入本地文件路径

## 技术栈

- HTML5
- CSS3 (CSS Custom Properties, Grid, Animations)
- Vanilla JavaScript (无框架)
- SVG Icons
- localStorage (数据持久化)
- QWeather API (天气数据)
- Notion API (数据集成)
